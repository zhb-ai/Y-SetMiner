## 报表分类

### 第一类：销售与客户（报表01-06）
核心销售类报表，全部基于物化视图，查询性能最优。

| 报表 | 核心分析维度 | 典型用途 |
|------|------------|---------|
| 01 销售业绩 | 时间/部门/业务员/品类 | 日报、月报、同比环比 |
| 02 客户价值 | 客户/地区/品类 | ABC分类、集中度 |
| 03 客户活跃度 | 客户/部门/业务员 | 流失预警 |
| 04 新客户开发 | 月度/业务员/地区 | 拓展评估 |
| 05 业务员绩效 | 业务员/部门 | 排名、同比 |
| 06 品类销售 | 品牌/大类/品类 | ABC分类、趋势 |

### 第二类：库存与物流（报表07-10）
库存管理类报表，部分基于物化视图，部分直查基表。

| 报表 | 数据来源 | 典型用途 |
|------|---------|---------|
| 07 库存监控 | MV_INVENTORY_SNAPSHOT | 库存总览、预警 |
| 08 库存周转 | MV_INVENTORY_SNAPSHOT + MV_SALES_DETAIL | 呆滞识别、周转率 |
| 09 出入库统计 | IC_GENERAL_H/B (基表) | 出入库流转 |
| 10 调拨在途 | TO_BILL/TO_BILL_B (基表) | 超期预警 |

### 第三类：财务与风控（报表11-14）
财务管控类报表，覆盖应收、信用、毛利、经营综合。

| 报表 | 核心功能 | 典型用途 |
|------|---------|---------|
| 11 应收账龄 | 账龄分布/超期预警 | 应收回款管理 |
| 12 客户信用 | 额度使用/超限预警 | 风险管控 |
| 13 毛利分析 | 品类毛利/商品毛利 | 盈利能力评估 |
| 14 经营日报 | 多维综合看板 | 日常经营决策 |

### 第四类：采购与供应链（报表15-16）
采购管理类报表，主要直查基表。

### 第五类：专项分析（报表17-19）
专项业务分析报表。

| 报表 | 状态 | 说明 |
|------|------|-----|
| 17 电商业务 | 待确认 | 需确认电商平台识别方式 |
| 18 放货业务 | 可用 | 已出库未开票监控 |
| 19 退货分析 | 可用 | 通过负数量识别退货 |

## 核心计算公式

```sql
-- 毛利率
ROUND(SUM(毛利) / NULLIF(SUM(销售流水), 0) * 100, 2)

-- 同比增长率
ROUND((本期值 - 去年同期值) / NULLIF(去年同期值, 0) * 100, 2)

-- 环比增长率
ROUND((本期值 - 上期值) / NULLIF(上期值, 0) * 100, 2)

-- 库存周转天数 (90天口径)
ROUND(90 * 库存金额 / NULLIF(近90天销售成本, 0), 0)

-- 退货率
ABS(退货流水) / NULLIF(正常销售流水, 0) * 100

-- 超期天数 (应收)
DATEDIFF(CURRENT_DATE(), 预计结算日期)

-- 信用额度使用率
已用额度 / NULLIF(总信用额度, 0) * 100
```

## 通用筛选条件

```sql
-- 赠品过滤（销售类报表必加）
AND 是否赠品 = 'N'

-- 时间范围
WHERE 年月 >= DATE_FORMAT(DATE_SUB(CURRENT_DATE(), INTERVAL 12 MONTH), '%Y-%m')  -- 近12月
WHERE 年度 = YEAR(CURRENT_DATE())  -- 本年
WHERE 日期 >= DATE_FORMAT(CURRENT_DATE(), '%Y-%m-01')  -- 本月

-- 基表标准过滤
AND pk_corp = '1006' AND dr = 0

-- 商品编码过滤
AND SUBSTR(inv.invcode, 1, 2) != '98'
```

## 注意事项

- 物化视图数据为 T+1（每日凌晨刷新），需实时数据请直查基表
- 退货通过 `销售数量 < 0` 识别，无独立退货标记字段
- 电商报表（17）需确认平台识别逻辑后调整 WHERE 条件
- 信用管理报表（12）的已用额度取自 ARAP_DJFB 未核销余额
- 库存周转分析（08）使用当前库存快照近似平均库存
