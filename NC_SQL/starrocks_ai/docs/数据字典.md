
# 此文档记录了用友NC系统核心表结构的主要字段和相互直接的引用关系，生成sql时需要严格按照数据字典内容关系生成。
数据库使用的是Starrocks数据库，sql必须符合其语法结构。
pk_corp 公司主键 默认公司主键为 1006，相关sql查询的主表需要增加此条件过滤无效公司数据。
dr 删除标志 默认0 为有效数据，1为删除数据，相关sql需要增加条件过滤删除的无效数据。

## 重要说明
1. 所有表名必须使用大写（如：BD_INVBASDOC，SO_SALE）
2. 不需要指定数据库名称前缀（如：nc5x.）
3. 表关联时必须同时过滤 pk_corp='1006' 和 dr=0
